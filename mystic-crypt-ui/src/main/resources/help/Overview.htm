Overview

This application provides a very simple search for finding duplicate files. It is simple to use. The user have to set only the two directories which will be compared. You can set five file attributes how the files are going to be compared.

Duplicate Files Finder Version 1.0

  • Search for files with the same or similar filename
  • Compare arbitrary files byte by byte
  • Find duplicate large VIDEO or MP-3 files that shrinking your harddrives and save money and time
  • Compare existing files collections and find doublets or even triplets

In companies administrators can seek the files from the employee for unnecessary file copies and redundant Backups.

The comminity edition is free to download. The enterprise edition have more features and can be purchased online.
